# burmese-agentic-website-template-workflow
A Next.js template designed for Myanmar local businesses that want a modern web presence without depending on a developer for every change. Built around an agentic content layer so business owners can update products, prices, and page content by talking to an AI agent (Claude Code, Codex CLI, or similar) — no code knowledge needed.

Includes a manual payment flow built for the Myanmar market (mobile banking number + screenshot upload), Telegram-integrated RAG chatbot running on a self-hosted LLM (Gemma 3 4B via Ollama), and a per-client isolated stack using Vercel, Supabase, Cloudflare R2, and Clerk.

Designed to be cloned once per business client, with clear boundaries on what the AI agent can and cannot touch.
